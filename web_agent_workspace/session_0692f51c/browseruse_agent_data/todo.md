# Verify Main Heading Text on Example.com

## Goal: To successfully navigate to example.com and extract the main heading text for validation.

## Tasks:
- [ ] Navigate to the URL https://example.com.
- [ ] Verify that the page loads successfully and the URL in the address bar is https://example.com.
- [ ] Locate the main heading element on the page (typically an <h1> tag).
- [ ] Extract the visible text content from the main heading element.
- [ ] Verify that the extracted main heading text is not empty.
- [ ] Record the extracted main heading text for further validation.
- [ ] If the page fails to load, report a "Page Load Failure" error.
- [ ] If the main heading element is not found, report a "Main Heading Element Not Found" error.
