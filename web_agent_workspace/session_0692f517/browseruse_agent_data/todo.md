# Verify Main Heading Text on Example.com

## Goal: To navigate to example.com and retrieve the text of the main heading (H1) element.

## Tasks:
- [ ] Navigate to https://example.com.
- [ ] Verify the page loads completely and the URL displayed in the browser is https://example.com.
- [ ] Locate the main heading element (e.g., <h1>).
- [ ] Extract the text content from the main heading element.
- [ ] Verify the extracted text is not empty and is a readable string.
- [ ] If the page fails to load or returns an error (e.g., 404, 500), report the page load failure.
- [ ] If the main heading element is not found on the page, report that the element is missing.
